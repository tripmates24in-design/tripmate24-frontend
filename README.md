# TravelSide

A modern, user-friendly travel recommendation website built with React 18, TypeScript, Vite, and Tailwind CSS.

## Features

- **Modern & Responsive Design**: built with Tailwind CSS, fully responsive for Mobile, Tablet, and Desktop.
- **Dynamic Search**: Filter destinations by price, category, and rating.
- **Interactive UI**: Smooth animations with Framer Motion, Swiper carousels, and engaging hover effects.
- **Detail Pages**: Comprehensive destination details with photo galleries, itineraries, and booking forms.
- **Mock Data**: Realistic mock data for destinations and reviews.

## Tech Stack

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **React Router v6**
- **Framer Motion**
- **Swiper.js**
- **React Hot Toast**
- **React Query**

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

3. **Build for Production**
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, etc.)
- `src/pages`: Main page components (Home, Search, DestinationDetail, etc.)
- `src/data`: Mock data for the application.
- `src/types`: TypeScript definitions.
